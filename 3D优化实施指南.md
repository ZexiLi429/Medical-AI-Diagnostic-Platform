### 修改3: medsam_service.py（后端返回mask_array）

**文件路径**: `MedSAM-main/medsam_service.py`

**修改目标**: 后端响应中添加二值掩码数组

**修改内容**:

```python
# ========== 找到 /segment 路由的返回部分（约第120-140行） ==========

# 原代码:
return {
    "image_url": f"/outputs/{filename}",
    "mask": base64_mask,
    "confidence": float(max_score),
}

# ✅ 修改为:
return {
    "image_url": f"/outputs/{filename}",
    "mask": base64_mask,
    "confidence": float(max_score),
    
    # 🆕 添加以下字段
    "mask_array": mask_binary.tolist(),  # 转换为列表供前端使用
    "area_mm2": calculate_area_mm2(mask_binary, pixel_spacing=0.5),  # 假设0.5mm/pixel
    "bbox_refined": get_precise_bbox(mask_binary).tolist(),
}

# ========== 在文件末尾添加辅助函数 ==========

def calculate_area_mm2(mask: np.ndarray, pixel_spacing: float = 0.5) -> float:
    """
    计算掩码实际面积(mm²)
    
    Args:
        mask: 二值掩码数组
        pixel_spacing: 像素间距(mm/pixel)
    
    Returns:
        面积(mm²)
    """
    pixel_count = np.sum(mask > 0)
    pixel_area_mm2 = pixel_spacing * pixel_spacing  # mm²
    total_area = pixel_count * pixel_area_mm2
    return float(total_area)


def get_precise_bbox(mask: np.ndarray) -> np.ndarray:
    """
    从掩码中提取精确边界框
    
    Args:
        mask: 二值掩码数组
    
    Returns:
        边界框 [x1, y1, x2, y2]
    """
    rows = np.any(mask, axis=1)
    cols = np.any(mask, axis=0)
    
    if not rows.any() or not cols.any():
        return np.array([0, 0, 0, 0])
    
    rmin, rmax = np.where(rows)[0][[0, -1]]
    cmin, cmax = np.where(cols)[0][[0, -1]]
    
    return np.array([cmin, rmin, cmax, rmax])
```

**预期效果**:
- 后端返回JSON包含完整的掩码数组数据
- 前端可以直接使用mask_array创建3D分割
- 增加面积和精确边界框信息

## 🐛 故障排查

### 问题1: 3D视图仍然卡顿

**可能原因**: 
- 设备检测失败，使用了错误配置
- WebGL硬件加速未启用

**解决方案**:
```javascript
// 在浏览器Console中运行:
import { getDeviceInfo } from './extensions/cornerstone/src/utils/deviceDetection';
console.log(getDeviceInfo());

// 检查输出中的:
// - capability: 应该是 'low', 'medium', 或 'high'
// - webglVersion: 应该是 'WebGL 2.0'
// - renderer: GPU型号

// 如果capability不正确，手动设置:
localStorage.setItem('forceDeviceCapability', 'low');
```

**启用Chrome硬件加速**:
1. chrome://settings/ → 搜索 "硬件加速"
2. 启用 "使用硬件加速模式（如果可用）"
3. 重启浏览器

### 问题2: SAM分割不显示在3D视图

**可能原因**:
- 后端未返回mask_array
- 3D viewport未初始化

**解决方案**:

1. 检查后端响应（Network标签）:
```json
// 应包含:
{
  "mask_array": [[0,0,1,1,...], ...],  // ✅ 存在
  "confidence": 0.95
}
```

2. 如果mask_array为空，检查 `medsam_service.py` 是否已修改

3. 如果前端报错 "无法获取viewport"，确保3D视图已打开:
```javascript
// 手动切换到包含3D视图的布局
commandsManager.runCommand('setViewportLayout', {
  layoutType: '3DMain'  // 或 '3DPrimary', '3DFourUp'
});
```
